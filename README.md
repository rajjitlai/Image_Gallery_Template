# Aesthetic Image Gallery Template

A modern, high-performance, and visually stunning image gallery template built with **Next.js 16**, **React 19**, and **Tailwind CSS 4**. This template features glassmorphism design, smooth scrolling, and beautiful scroll-reveal animations.

## ✨ Features

- 🚀 **Next.js 16 (App Router)** - Leveraging the latest features and Turbopack.
- ⚛️ **React 19** - Built with the latest stable version of React.
- 🎨 **Tailwind CSS 4.0** - Utilizing the newest version of Tailwind for styling.
- ✨ **Glassmorphism UI** - Modern frosted-glass card components.
- 🌑 **Dark Mode by Default** - Optimized for a sleek dark aesthetic.
- 🎢 **Smooth Scrolling** - Powered by **Lenis** for an inertial scrolling experience.
- 🎬 **Reveal Animations** - Staggered scroll animations using **Framer Motion**.
- 🖼️ **Optimized Images** - Blur-to-focus loading effect using `next/image`.
- 📱 **Fully Responsive** - Seamless experience across mobile, tablet, and desktop.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll:** [Lenis](https://lenis.darkroom.engineering/)
- **Utility:** [clsx](https://github.com/lukeed/clsx) & [tailwind-merge](https://github.com/dcastil/tailwind-merge)

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm, yarn, or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rajjitlai/Image_Gallery_Template.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Image_Gallery_Template
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
